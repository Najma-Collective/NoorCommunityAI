Understanding the technology behind personalized language learning

Interactive AI-Powered Learning System

Our AI Instructor provides personalized pre-class instruction and scaffolded support for language development, designed specifically for Palestinian university students learning English for Academic Purposes.

Key Features

  • Task-Based Learning (TBL) Methodology: Lessons are structured around meaningful communicative tasks that mirror real-world academic and professional scenarios.
  • Trauma-Informed Pedagogy: Every interaction prioritizes psychological safety, choice, and empowerment, creating a supportive learning environment.
  • Bilingual Support: Strategic use of translanguaging (English-Palestinian Arabic) to enhance comprehension and build cultural rapport.
  • Adaptive Pathways: The system adjusts to student responses, offering differentiated support and pacing based on individual needs.
  • Privacy-First Design: Absolute prohibition on collecting or processing personally identifiable information.
  • Comprehensive Feedback: Multi-layered formative assessment using recasting, guided discovery, and metacognitive reflection.

Pedagogical Principles

The system is grounded in research-based second language acquisition principles:

  • Zone of Proximal Development: Scaffolding that provides "i+1" input‚Äîjust beyond current ability.
  • Retrieval Practice: Spaced repetition and active recall to strengthen long-term retention.
  • Output Maximization: Student-centered design that prioritizes learner talk time over system exposition.
  • Metacognition: Explicit focus on learning strategies, self-assessment, and reflective practice.

Technical Architecture

The AI tutor is powered by Google's Gemini 2.5 Flash model with a structured XML system prompt that ensures consistent, pedagogically-sound interactions. The system employs:

  • Real-time conversational AI with context retention across lesson stages
  • Visual interface with semantic HTML components for enhanced learning
  • Browser-based speech synthesis for pronunciation modeling
  • Vocabulary tracking and session reporting for learner reflection
  • Portable prompt architecture compatible with various AI models
